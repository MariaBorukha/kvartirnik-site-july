# kvartirnik-site-july

Квартирник у Маши Боруха — сайт с автоматическим движком событий (июльская версия).

Место и время, где музыканты-любители играют и поют. Москва, с 2022 года. Бесплатный проект.

🌐 **Сайт:** https://mariaborukha.github.io/kvartirnik-site-july/ *(появится после публикации через GitHub Pages)*

## Чем отличается от старого сайта

Старый репозиторий [`kvartirnik-2`](https://github.com/MariaBorukha/kvartirnik-2) — весь контент зашит прямо в HTML, каждое событие правится вручную. Здесь весь «переменный» контент вынесен в **один файл `data/events.json`**, а страницы событий собираются из него автоматически. Добавить новый квартирник = дописать один блок в JSON. Подробности — в [PLAYBOOK.md](PLAYBOOK.md).

## Как это работает

Страница следующего события сама «превращается» в страницу прошедшего: пока дата в будущем — показывается анонс и регистрация; после события в ту же запись добавляются фото, песни и ссылка на Google Диск, и она уезжает в архив.

```
events.html (следующее)  →  archive.html (список прошедших)  →  event.html?id=… (одно событие)
                    всё рендерится из data/events.json движком js/app.js
```

## Файлы

| Файл | Описание |
|---|---|
| `index.html` | Главная страница |
| `events.html` | Следующее мероприятие (из JSON) |
| `event.html` | Страница одного события — `event.html?id=slug` (из JSON) |
| `archive.html` | Архив прошедших квартирников (из JSON) |
| `playlist.html` | Плейлист и предложение песни |
| `team.html` | Команда квартирника |
| `finance.html` | Финансовая отчётность |
| `rules.html` | Правила поведения |
| `styles.css` | Общие стили всех страниц |
| `js/app.js` | Движок: читает `events.json` и рисует страницы событий |
| `data/events.json` | **Единственный файл, который правим для контента** |
| `PLAYBOOK.md` | Как устроен сайт и как за 5 минут добавить событие |
| `CMS-BACKLOG.md` | План простой CMS для редактирования без кода |

## Добавить новый квартирник

Открыть `data/events.json` → добавить блок в начало массива `events` → закоммитить и запушить. Пошагово — в [PLAYBOOK.md](PLAYBOOK.md).
